# TodoList

A modern, minimalist task management application built with React and Vite. This project allows users to manage their daily tasks with a clean and responsive user interface.

## Description

TodoList is a single-page application designed to help users organize their activities. It features a straightforward interface for adding, tracking, and deleting tasks. The design focuses on usability and minimalism, utilizing a distraction-free layout.

## Features

- **Task Management**: easy addition of new tasks to the list.
- **Completion Tracking**: toggle tasks as completed with a visual strikethrough effect.
- **Task Deletion**: remove unwanted tasks from the list permanently.
- **Minimalist Design**: a clean, centered layout with modern styling and smooth interactions.
- **Responsive Interface**: adapts to different screen sizes.

## Technlogies Used

- **React**: for building the user interface components.
- **Vite**: for fast development tooling and bundling.
- **CSS**: for custom, responsive styling without external frameworks.

## Installation and Setup

Follow these steps to run the project locally on your machine.

### Prerequisites

- Node.js installed on your system.

### Steps

1.  **Clone the repository**
    ```bash
    git clone https://github.com/marioroman2005/todoList.git
    cd todoList
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```

3.  **Run the development server**
    ```bash
    npm run dev
    ```

4.  **Open the application**
    Open your web browser and navigate to the URL shown in the terminal (usually `http://localhost:5173` or `http://localhost:3000`).

## Project Structure

- `src/components`: contains reusable UI components like `TodoForm`, `TodoItem`, and `TodoList`.
- `src/App.jsx`: main application component and logic.
- `src/App.css`: global and component styles.
